# 布局组件

> 🚧 **页面建设中** - 该页面正在完善中，敬请期待！

DVHA Pro 提供了丰富的布局组件，帮助您快速构建企业级管理界面。

## 组件列表

### DuxLayout

主要的应用布局组件，包含侧边栏、头部、内容区域等。

### DuxAuthLayout

认证相关的布局组件，用于登录、注册等页面。

### DuxPageLoading

页面加载时的布局组件。
